This is a collection of documents related to Terraform, a IaC (infrastructure as a code) system for automatic management of cloud platforms using code.

# Introduction
Terraform is a IaC (infrastructure as a code) system which allows us to create and manage cloud resources using code. 

We write a code which designates which resources should be created, deleted or modified, and after executing it, Terraform will execute all the actions which we coded.
