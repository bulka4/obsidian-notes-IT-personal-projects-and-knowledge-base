# Introduction
A conditional task is used for:
- Skipping ([[Airflow - Skipping tasks in a DAG|link]]) - deciding whether or not to skip all the further, downstream tasks in the DAG
- Branching ([[Airflow - Branching tasks in a DAG|link]]) - deciding which task to execute next in the DAG
