# Introduction
In the `powershell_scripts` folder there are powershell scripts which can be used for working with Docker:

`docker_compose_up.ps1` 
Build and run a Docker container and save logs in the `dockerfiles/docker_logs` folder (that script needs to be ran from the repository root)

`docker_clean.ps1` 
Delete all the Docker containers and images

`sql_connect.ps1 -password <your_password>` 
Connect to the created MS SQL server
