# Introduction
Counting measure is counting number of elements in a set.
# Definition
Let $(\mathcal{X}, \mathcal{B}_{\mathcal{X}})$ be a measurable space ([[Measurable space|link]]). The counting measure $\upsilon$ on $\mathcal{X}$ is defined as:
$$
\upsilon(A) = 
\begin{cases}
 |A|, \text{ if A is finite} \\
 \infty, \text{ otherwise}
\end{cases}
$$
for every $A \in \mathcal{B}_{\mathcal{X}}$.
