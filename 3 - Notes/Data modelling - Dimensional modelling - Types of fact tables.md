# Introduction
There are 3 major types of fact tables:
## a) Transaction facts
- One row per event
- Example: each purchase
## b) Snapshot facts
- Periodic state
- Example: daily account balance
## c) Accumulating snapshot facts
- Tracks process lifecycle
- Example: order → shipped → delivered
