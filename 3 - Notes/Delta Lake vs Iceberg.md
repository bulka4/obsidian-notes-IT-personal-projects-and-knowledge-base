# Key differences
Iceberg compared to Delta Lake:
- Supports more engines (not only Spark but also Flink, Trino, Presto, Hive and Impala)
- Is more efficient at storing metadata, performing updates and deletes at large tables
- Is fully open-source while Delta Lake has a commercial version with additional features
