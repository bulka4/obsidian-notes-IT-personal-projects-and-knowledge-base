# Introduction
Numbers can be written in an exponential form:
$$
\large
+ / - 1 \cdot \text{fraction} \cdot 10^\text{exponent}
$$
where:
- $+/- 1$ - Determines a sign of a number
- Fraction:
	- A number between 1 and 2
	- Determines what digits the number consist of
- Exponent:
	- To which power raise a number 10
	- It determines which digits goes to decimal places

So for example, we can write a number in an exponential notation like this:
$$
123.456 = 1 \cdot 1.23456 \cdot 10^2
$$
So here we have:
- $1$ - A positive sign
- $1.23456$ - Collection of digits we have in the number (1, 2, 3, 4, 5, 6)
- $10^2$ - Decimal places start after the third digit (after the digit 3)
