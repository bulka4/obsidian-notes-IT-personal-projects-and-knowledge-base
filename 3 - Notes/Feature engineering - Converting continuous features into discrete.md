# Replacing a continuous values with ranges
We can replace values of a continuous variable with ranges which this value belongs to. For example:
- 1 -> $[1, 5]$
- 7 -> $[5, 10]$
- 13 -> $[10, 15]$

Different ranges becomes classes (possible values of a categorical variable).
# Clustering
We can use clustering in order to create groups (clusters) where each group contains samples with a specific range of values of a continuous variable.
