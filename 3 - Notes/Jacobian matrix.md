# Introduction
Jacobian matrix represents values of derivatives of a function for a specific argument.

Depending on what is the type of the function's input and output (matrix, vector or scalar), we have differently defined Jacobian matrix.

More information about them can be found in the documents below:
- Input - vector & output - scalar: [[Jacobian matrix - Input - vector & output - scalar|link]]
- Input - vector & output - vector: [[Jacobian matrix - Input - vector & output - vector|link]]
- Input - matrix & output - scalar: [[Jacobian matrix - Input - matrix & output - scalar|link]]
- Input - matrix & output - vector: [[Jacobian matrix - Input - matrix & output - vector|link]]
