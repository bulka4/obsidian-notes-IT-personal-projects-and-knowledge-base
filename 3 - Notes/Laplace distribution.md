# Introduction
We say that a random variable $X$ ([[Random variable|link]]) has the Laplace distribution if its probability distribution function ([[Probability distribution (density, mass) function|link]]) is defined as:
$$
p_X(x) = \frac{1}{2b}
\exp(-\frac{|x - \mu|}{b})
$$
And we note it as:
$$
\begin{gather}
X \sim \mathcal{L}(\mu, b) \\
\text{or} \\
X \sim \text{Laplace}(\mu, b)
\end{gather}
$$
