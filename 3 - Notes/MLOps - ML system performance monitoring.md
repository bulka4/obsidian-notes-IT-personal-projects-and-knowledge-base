# Introduction
ML system performance monitoring is about observing how well a system which makes predictions using a ML models performs. 

We can monitor things like:
- Prediction latency (how long does it take to make a single prediction)
- Predictions throughput (number of predictions per second)
- Error rate (percentage of failed predictions. It is not about inaccurate predictions but when prediction is not make at all)
