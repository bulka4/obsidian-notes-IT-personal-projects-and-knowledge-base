# Introduction
When it comes to MLOps, we can monitor two things:
- Model performance ([[MLOps - Model performance monitoring & automatic model update|link]]) - how good are the predictions (accuracy etc.)
- ML system ([[MLOps - ML system performance monitoring|link]]) - How well the system which uses a ML model performs (prediction latency, throughput etc.)
