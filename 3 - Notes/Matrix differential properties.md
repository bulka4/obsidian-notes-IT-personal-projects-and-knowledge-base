# Introduction
This document describes common properties related to matrix derivatives.
# vec–Kronecker derivative identity
For matrices $W \in \mathbb{R}^{m \times n}, a \in \mathbb{R}^{n \times 1}$, we have
$$
\frac{\partial (Wa)} {\partial \text{vec}(W)} = a^T \otimes I
$$
