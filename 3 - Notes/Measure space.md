# Introduction
Let $X$ be a set ([[Set|link]]) and $\mathcal{F}$ be a $\sigma$-algebra ([[Sigma algebra|link]]) and $\mu$ be a measure ([[Measure|link]]). The triple $(X, \mathcal{F}, \mu)$ is called a measure space.
