# Introduction
Modulo is an operation which returns a reminder of a division. We say $x$ modulo $y$, denote that as $x \text{ mod } y$.

Module is such a number $r$ :
$$
x \text{ mod } y = r
$$
that:
- $r \in \mathbb{Z}$ 
- $x = ny + r$ for some $n \in \mathbb{Z}$ 
- $|r| < |y|$ 
- $y \neq 0$
