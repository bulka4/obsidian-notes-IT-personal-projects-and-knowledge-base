# Introduction
We say that a random variable $X$ ([[Random variable|link]]) has the normal (Gaussian) distribution if its probability distribution function ([[Probability distribution (density, mass) function|link]]) is defined as:
$$
p_X(x) = \frac{1}{\sqrt{2 \pi \sigma^2}}
\exp(-\frac{(x - \mu)^2}{2 \sigma^2})
$$
And we note it as:
$$
X \sim \mathcal{N}(\mu, \sigma^2) 
$$
