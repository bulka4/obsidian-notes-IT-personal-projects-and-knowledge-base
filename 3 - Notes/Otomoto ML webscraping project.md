# Introduction
In this project, we:
- Create a dataset about used cars offers from otomoto.pl using webscraping with the BeautifulSoup Python library
- Build a model for predicting a price of a car
# Code repository
Repository with project's code - [github.com](https://github.com/bulka4/otomoto_ml_webscraping).
# Predicting a price of a car
We predict a price of a car using a linear regression.

We program it using a function from `sklearn`.
