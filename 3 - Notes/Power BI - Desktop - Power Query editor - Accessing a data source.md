# Power Query editor - Running SQL queries
To be able to access data source from the desktop app, we need to go to the `Home > Transform data`. That opens Power Query editor where we can see SQL queries used to prepare data for the dashboard and see results.
# Credentials
In the Power Query Editor, we will asked for credentials and we can provide them there.

We can also provide credentials in the data source settings: `File > Options and settings > Data source settings`.
