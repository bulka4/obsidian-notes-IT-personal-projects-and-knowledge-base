## Random_state argument

It determines how we shuffle a dataset, if random_state = none then each time when we shuffle dataset this value is random and different, and we shuffle a dataset in a different way. When we set random_state equals to some integer then each time we shuffle dataset it will be shuffled in the same way. For different values of random_state a dataset is shuffled in a different way.
