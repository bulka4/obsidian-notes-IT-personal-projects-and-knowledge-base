# Introduction
Official documentation: [docs.ray.io](https://docs.ray.io/en/latest/cluster/kubernetes/getting-started.html).

KubeRay is Kubernetes operator enabling running Ray on Kubernetes.

It provides 3 CRDs:
- RayCluster – Used to create a Ray cluster
- RayJob – Used to run batch jobs on a Ray cluster
- RayService ([[KubeRay - RayService|link]]) – Used to create a Ray cluster and run on it Ray Serve apps
