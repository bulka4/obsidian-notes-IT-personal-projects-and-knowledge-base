Residuals are a difference between our model's predictions and real values we were trying to predict. We calculate them using the formula:
$$
e_i = y_i - \hat{y}_i
$$
where:
- $e_i$ - Residual for the i-th data sample
- $y_i$ - The real value for the i-th data sample
- $\hat{y}_i$ - Prediction of the real value for the i-th data sample# Variance & Bias
