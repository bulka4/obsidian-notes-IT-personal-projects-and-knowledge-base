# Introduction
This document describes common operations we can perform on sets and how to denote them:
- Union: $A \cup B = \{x \mid x \in A \vee x \in B \}$ 
- Intersection: $A \cap B = \{x \mid x \in A \wedge x \in B \}$ 
- Difference: $A \setminus B = \{x \mid x \in A \wedge x \notin B \}$ 
- Complement: $A^c = U \setminus A$ ($U$ - universal set)
- $\bigcup_{i=1}^n A_i = A_1 \cup \ldots \cup A_n$ 
- $\bigcap_{i=1}^n A_i = A_1 \cap \ldots \cap A_n$ 
- $2^X$ - Collection of all subsets of the set $X$
