#MathematicalAnalysis #Mathematics 

# Formal definition
For a given set ([[Set|link]]) $X$, we say that **$\sigma$-algebra generated** by a collection $\mathcal{C}$ of subsets of $X$ is the smallest $\sigma$-algebra ([[Sigma algebra|link]]) that contains those sets, that is:
$$
\sigma(\mathcal{C}) = \bigcap\{\mathcal{F} \subseteq 2^X :\ \mathcal{F} \text{ is a } \sigma \text{-algebra and } \mathcal{C} \subseteq \mathcal{F} \}
$$
where $2^X$ indicates a collection of all subsets of $X$.
