#MathematicalAnalysis #Mathematics 

# Formal definition
A measure space ([[Measure space|link]]) $(\mathcal{X},\ \mathcal{F},\ \mu)$ is $\sigma$-finite if
$$
\large
\exists \{A_i\}_{i=1}^\infty \subseteq \mathcal{F}:\ X = \bigcup_{i=1}^\infty A_i \wedge (\mu(A_i) < \infty,\ \forall i)
$$
