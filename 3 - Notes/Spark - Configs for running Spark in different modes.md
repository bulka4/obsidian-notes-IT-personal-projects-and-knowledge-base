# Introduction
In what mode ([[Spark - Modes|link]]) Spark will run, depends on what client we use to run a Spark app ([[Spark - Clients starting calculations|link]]) and what configurations we use.

To learn more about how to choose a mode in which Spark will run depending on which client we use to run the Spark script, refer to this document - [[Spark - Setting up a mode when using different clients]].
