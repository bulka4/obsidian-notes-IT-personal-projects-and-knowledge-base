# Introduction
We can use Prometheus, Grafana and Spark History Server to monitor our Spark jobs.

Prometheus will gather metrics about Spark Jobs.

Grafana will create visual dashboards using metrics gathered by Prometheus.

Spark History Server gives more detail about Spark jobs.
