# Introduction
In what mode ([[Spark - Modes|link]]) Spark will run, depends on what client we use to run a Spark app ([[Spark - Clients starting calculations|link]]) and what configurations we use.

To learn more about what configs to use when running Spark in different modes, refer to this document - [[Spark - Configs for running Spark in different modes]].

Below documents describe how to choose a mode in which Spark will run depending on which client we use to run the Spark script:
- Spark submit - [[Spark - Setting up a mode with Spark submit]]
- Python - [[Spark - Setting up a mode when running a script with Python]]
