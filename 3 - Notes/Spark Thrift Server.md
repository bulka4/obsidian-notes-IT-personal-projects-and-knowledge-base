# Introduction
Spark Thrift Server is used to create a long-running Spark driver ([[Spark - Components - Master, driver and executors|link]]) and session ([[SparkSession|link]]) which can be then used in jupyter notebooks or by other applications to perform calculations using Spark by connecting through JDBC/ODBC (for example by dbt ([[dbt with Spark|link]])).
