# Introduction
A tensor is a general term which can refer to a set on numbers in any amount of dimensions, so tensor can be:
- A number (0-D tensor)
- A vector (1-D tensor)
- A matrix ([[Matrix|link]]) (2-D tensor)
- Numbers in higher dimensions (4-D, 5-D, ... tensors)
