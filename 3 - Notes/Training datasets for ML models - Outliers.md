# Introduction
Outlier is a data sample ([[Dataset of observations of random variables X, Y - Variables per sample, feature and target variable|link]]) which feature values are significantly different than other data samples within a dataset.

It is useful sometimes to remove outliers from a training dataset because thanks to this, it might be easier for the model to capture patterns which are common because those outliers might not match those patterns.

Because of that, model can be worse when it comes to detecting such outliers but overall performance will be better since number of outliers is very small.
