#Probability #Mathematics 

# Introduction
Variance measures how spread out are values of a random variable, that is what is an average difference between a value of a random variable and its mean.
# Definition
Variance of a random variable $X$ is defined as:
$$
\text{Var}(X) = \mathbb{E}[X - \mathbb{E}(X)]^2
$$
