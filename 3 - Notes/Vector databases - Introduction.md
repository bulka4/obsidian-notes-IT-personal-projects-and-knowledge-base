#VectorDatabases

# Introduction
There are two ways how vector dbs ([[_Vector_databases|link]]) can store their data:
- In memory
- On disk

Example tools for creating vector dbs:
- Milvus ([[Milvus|link]]) – Stores data on disk
- FAISS – Stores data only in memory
- Chroma – Can store data in memory or on disk
