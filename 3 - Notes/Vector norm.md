# Introduction
A norm is a function that assigns non-negative length to each vector ([[Vector|link]]) $x$ in a vector space $V$, satisfying below properties for any vector $x, y \in V$ and scalars $\alpha$ :
$$
\begin{align}
& \text{1) } ||x|| \ge 0 \\
& \text{2) } ||x|| = 0 \iff x = 0 \\
& \text{3) } ||\alpha x|| = |\alpha| \cdot ||x|| \\
& \text{4) } ||x + y|| \le ||x|| + ||y||
\end{align}
$$
# Examples
Common examples include:
- Euclidean norm (L2):
$$
||x||_2 = \sqrt{x_1^2 + \dots + x_n^2}
$$
- Manhattan norm (L1):
$$
||x||_1 = |x_1| + \dots + |x_n|
$$
- Max norm:
$$
||x||_\infty = \max(|x_1|, \dots, |x_n|)
$$
