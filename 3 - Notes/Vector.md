# Introduction
A vector is an ordered list on real numbers:
$$
x = (x_1, \ldots, x_n) \in \mathbb{R}^n
$$
# Column and row vectors
In general, a vector is not a column neither a row. But usually, we interpret a vector either:
- As a single column matrix ([[Matrix|link]]):
$$
x = \begin{pmatrix}
x_1 \\
\vdots \\
x_n
\end{pmatrix}
\in \mathbb{R}^{n \times 1}
$$
	which we call a **column vector.**
- Or as a single row matrix:
$$
x = \begin{pmatrix}
x_1 & \dots & x_n
\end{pmatrix}
\in \mathbb{R}^{1 \times n}
$$
	which we call a **row vector**.
