# Introduction
We can use powershell scripts from the powershell_scripts folder:
- `docker_build_run.ps1` - build and run Docker container
- `sql_connect.ps1 -password <your_password>` - connect to the created MS SQL db
- `docker_clean.ps1` - remove all the Docker containers and images.
