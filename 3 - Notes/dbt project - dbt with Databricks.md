# Enabling a Unity Catalog
We need to enable a Unity Catalog in our Databricks workspace so we can create it and use in our dbt project. It is described here: [learn.microsoft.com](https://learn.microsoft.com/en-us/azure/databricks/data-governance/unity-catalog/get-started#auto-enabled-check).
