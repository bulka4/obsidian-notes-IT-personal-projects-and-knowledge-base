# Introduction
In this repository are files used by Docker to create and configure a MS SQL server and dbt project which is performing transformations on that MS SQL server.

Here is a high level overview of what files and folders we have here:
- **mssql_preparation** - Files needed for Docker for prepare a MS SQL server where dbt will be making data transformations.
- **powershell_scripts** - Useful powershell scripts. More info in the 'Powershell scripts' section of this documentation.
- **data_warhouse** - dbt project. We have there macros, models and snapshots.
