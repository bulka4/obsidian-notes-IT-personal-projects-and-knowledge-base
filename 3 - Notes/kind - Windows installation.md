# Introduction
- Download the 'kind-windows-amd64' file from the 'Assets' section on this website - [link](https://github.com/kubernetes-sigs/kind/releases) 
- Rename the downloaded file into 'kind.exe'
- Move this file into a chosen folder (for example `C:/ProgramFiles/kind`)
- Add to the `$PATH` env var this path (to edit this env var, search for 'environment variables' in the windows search bar)
